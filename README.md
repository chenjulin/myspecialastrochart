# 我的星盤網站 myspecialastrochart 🌟

這是一個使用 React + TypeScript 製作的個人星盤查詢網站。

## 🚀 如何使用

1. 安裝依賴：
```
npm install
```

2. 啟動本地開發伺服器：
```
npm start
```

3. 建置並部署：
```
npm run build
```

## ☁️ 部署到 Vercel

1. 將此專案上傳至 GitHub
2. 登入 Vercel 並連接 GitHub 帳號
3. 點選 `Import Project`，選擇此 Repo
4. 設定：
   - Build Command: `npm run build`
   - Output Directory: `build`
5. 點 Deploy 完成 🎉
